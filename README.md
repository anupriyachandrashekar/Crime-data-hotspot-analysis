# Crime Data Analysis & Hotspot Mapping

## Overview

This project analyzes urban crime data to identify spatial and temporal patterns and detect high-risk zones using data-driven techniques.


## Objectives

* Identify crime hotspots across different locations
* Analyze time-based crime patterns
* Support better decision-making for urban safety


## Tools & Technologies

* Python (pandas, NumPy, scikit-learn)
* Data Visualization libraries (Matplotlib / Seaborn)
* Clustering: K-Means


## Key Work

* Performed exploratory data analysis on crime datasets
* Applied K-means clustering to identify high-risk zones
* Conducted temporal analysis to detect peak crime periods
* Combined spatial and time-based insights for better interpretation


## Results

* Identified high-risk locations and time windows
* Generated insights useful for law enforcement planning
* Improved understanding of urban crime patterns
